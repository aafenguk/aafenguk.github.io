Nodejs学习笔记 - tls模块中的完美前向安全

最近在翻译Node.js中有机会系统学习了一下完美前向安全。

在tls模块的数据传输中，采用了Perfect Forward Secrecy，也就是完美前向安全。说的简单一点，就是在每个会话中会根据主密码生成临时密码，在整个会话过程中使用临时密码进行数据传输。这样即使主密码泄露，也不会泄露会话密钥，从而保护过去的通信数据。

Vue配置时的一个坑及解决

在VM中创建了一个Vue的项目，然后运行：

npm run dev

系统提示已经成功运行在8080端口，但是从Host的浏览器访问，总是提示无法访问。

检查了VirtualBox中的Port forwarding，也没有问题。

尝试了各种可能后终于找到解决办法了。需要更新文件： build/webpack.dev.conf.js， 将其中的

host: HOST || config.dev.host

替换为：

host: 0.0.0.0

再重新启动APP，问题解决！
